\# Data Folder



This project uses internal policy documents and compliance scenarios.



⚠️ The actual datasets are intentionally excluded from this repository 

to protect confidentiality and organizational privacy.



\## Expected Data Files (Local Only)



\- policy\_clauses\_cleaned.xlsx

\- compliance\_scenarios\_expanded.xlsx



\## Structure



\- Each policy clause includes:

&nbsp; - policy\_type

&nbsp; - section

&nbsp; - clause text



\- Each scenario includes:

&nbsp; - scenario description

&nbsp; - compliance label

&nbsp; - risk level



Developers can recreate sample data following the schema used in the code.
